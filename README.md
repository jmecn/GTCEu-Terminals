### A Lightweight GregTech Modern Addon

### What makes it so advanced?
The Advanced Terminal is an upgrade to the GregTechCEu Modern Terminal with several Quality of Life features including:
- Replacing Coils
- Replacing Input & Output Hatches/Buses
- Replacing Muffler Hatches, Maintenance Hatches, Parallel Hatches, Energy Hatches, and More
- Copying and Pasting Whole Multiblocks
- Saving Multiblock Schematics
- AE2 Inventory Connection
- Configurable Features

Special Thanks to Haze Vista, For providing textures and ideas.

(Note: AE2 Inventory Connection will be implemented later, including Parallel Hatches)
